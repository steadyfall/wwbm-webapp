No integration tests existed for the `register` view (`auth/views.py:10-41`). This PR adds a test module covering all major registration paths.

### Tests
Added `tests/unit/test_auth_views.py` — 7 integration tests using `django.test.Client` and `@pytest.mark.django_db`:

- Mismatched passwords → error message, redirect to `/auth/register/`
- Password contains username → error message, redirect to `/auth/register/`
- Missing required field (`password2`) → error message, redirect to `/auth/register/`

All 7 tests pass (`uv run pytest tests/unit/test_auth_views.py`).
